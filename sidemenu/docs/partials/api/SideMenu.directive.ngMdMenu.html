ngMdMenu
directive in module SideMenu

Description

this would list all menu items

Usage

as element:
<ng-md-menu
       ng-model="{array}"
       ng-filter="{string}">
</ng-md-menu>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelarray

Array for list Menu Items

ngFilterstring

To filter the menu items
