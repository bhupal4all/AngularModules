ngMdMenuGroup
directive in module SideMenu

Description

This is group menu which contains list of items

Usage

as element:
<ng-md-menu-group
       ng-model="{array}"
       ng-show="{boolean}"
       ng-open="{boolean}"
       ng-filter="{string}">
</ng-md-menu-group>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelarray

Array for list Menu Items

ngShowboolean

To show the menu group

ngOpenboolean

To the menu in expand mode

ngFilterstring

To filter the menu items
