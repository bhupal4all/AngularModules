ngMdSideMenu
directive in module SideMenu

Description

This is contins section of menu

Usage

as element:
<ng-md-side-menu
       ng-model="{array}"
       ng-side-menu-open="{boolean}"
       ng-show-loading="{boolean}"
       ng-is-error="{boolean}">
</ng-md-side-menu>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelarray

Array for list Menu Items

ngSideMenuOpenboolean

To show entire menu

ngShowLoadingboolean

To show loading messsage, if this sets, then menu wont be displayed

ngIsErrorboolean

is there any error while loading the menu and results empty
